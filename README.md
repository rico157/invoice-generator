## About The Project

Generate a simple invoice pdf with React

![Invoice Generator Screen Shot][product-screenshot]

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/rico157/invoice-generator.git
   ```
2. Install Yarn/NPM packages
   ```sh
   npm install or yarn
   ```
3. Run the app
   ```sh
   npm start or yarn start
   ```
